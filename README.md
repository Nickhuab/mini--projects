# Python Mini Projects

This repository contains beginner Python CLI projects to practice core programming concepts.

## Projects

### 1. Contact Book
- Add, search, delete contacts
- Uses dictionary and loops

### 2. Number Guessing Game
- User guesses a random number
- Provides hints (too high / too low)

### 3. Rock Paper Scissors
- Play against computer
- Uses random and condition logic

## How to Run

Make sure Python is installed, then run:

```bash
python contact_book.py
