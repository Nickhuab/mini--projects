# Python Mini Projects

This repository contains beginner-to-intermediate Python CLI projects focused on building core programming skills and logical thinking.

---

## Projects

### 1. Contact Book

* Add, search, delete, and view contacts
* Uses dictionary for data storage
* Demonstrates loops and condition handling

---

### 2. Number Guessing Game

* User guesses a randomly generated number
* Provides hints (too high / too low)
* Uses random module and loop control

---

### 3. Rock Paper Scissors

* Play against computer
* Uses random choices and conditional logic

---

### 4. Quiz App

* Multiple questions with score tracking
* Questions are shuffled to avoid repetition
* Case-insensitive answer checking
* Exit option to quit anytime

---

## ▶️ How to Run

Make sure Python is installed on your system.

Run any project using:

```bash
python filename.py
```

Example:

```bash
python contact_book.py
```

---

## Concepts Covered

* Loops (`for`, `while`)
* Conditionals (`if-elif-else`)
* Data structures (lists, dictionaries, tuples)
* Functions (basic usage)
* Random module
* Input handling
